## ğŸ› Troubleshooting

### Names Not Showing?
1. Wait 2-3 seconds after registration
2. Refresh the page
3. Check you're connected to the right wallet
4. Open browser console for errors

### Registration Failed?
1. Ensure you have enough USDC (>0.1)
2. Check name is 3-32 characters
3. Use only a-z, 0-9, and hyphens
4. Name can't start/end with hyphen

### Contract Not Found?
1. Check `.env` file has correct address:
   ```
   VITE_REGISTRY_ADDRESS=0x7F0768f96b2bc1CE19ED27D14c664078BBde3f19
   ```
2. Restart dev server:
   ```bash
   npm run dev
   ```
